# Machine Learning Tutorial: Random Forests
This repository contains a tutorial and Python implementation of the Random Forest machine learning technique for my MSc Data Science assignment.

## Contents
* *Individual Assignment ML RF.pdf*: The full tutorial document explaining the technique.
* *rf_code_tutorial.ipynb*: The Jupyter Notebook containing the code and implementation steps.

## How to Use
1. Download the PDF to read the conceptual overview and tutorial.
2. Open the .ipynb file in Jupyter Notebook, VS Code, or Google Colab to run the code.

## References

*Sun, Z., Wang, G., Li, P., Wang, H., Zhang, M. and Liang, X. (2024) ‘An improved random forest based on the classification accuracy and correlation measurement of decision trees’, Expert Systems with Applications, 237, p. 121549.
